# 🚀 TaskFlow Pro

A modern, premium-quality **Task Management Web App** built using **pure HTML, CSS, and JavaScript** in a single file. Designed with a startup-level UI/UX, smooth animations, glassmorphism effects, and smart productivity features.

---

## ✨ Features

### 🧠 Core Task Management

* ➕ Add new tasks instantly
* ✏️ Edit tasks inline
* 🗑 Delete tasks with smooth animation
* ✅ Mark tasks as completed / pending
* 📂 Task categories (Work, Study, Personal, Important)
* 📅 Due date & time support

### 🔍 Productivity Tools

* 🔎 Real-time task search
* 🎯 Filters: All / Active / Completed / Overdue
* 📊 Animated progress bar
* 🕒 Live real-time clock
* 💬 Auto-changing motivational quotes

### ⏰ Smart Reminder System

* Pleasant alert sound when task time arrives
* Animated popup notification
* Options:

  * ✅ Complete Now
  * 😴 Snooze 5 Minutes
  * ❌ Close

### 🎨 Premium UI/UX

* Glassmorphism design
* Transparent water-style buttons
* Smooth hover & click animations
* Floating cards animation
* Fully responsive (Mobile + Desktop)
* 🌙 Dark Mode / ☀️ Light Mode

### 💾 Storage

* Uses Local Storage
* Tasks remain saved after refresh

---

## 🛠 Tech Stack

* HTML5
* CSS3
* JavaScript (Vanilla JS)
